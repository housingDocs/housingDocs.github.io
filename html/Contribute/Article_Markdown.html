Tables
Table Structure
A Markdown Table consists of Rows and Columns. A table can contain multiple rows, which contain independent columns. Each Cell is separated by a Pipe Symbol "|". The first cell in a column, which is in the first row is the Title Cell.
Title Cell of Column 1 | Some Value | Another Value
Title Cell of Column 2 | Value spreads over entire row
New Row this time
And the Columns have changed in width
Yet another Value
Tables in Markdown
To create a Table in Markdown, use the <table> element. Then insert as many rows as you want using the <row> element. Columns also need a set column width which has to have a Percent Value.

For example a column with a width of 50% in markdown is:
<table>
    <row>
        <column width="50px">
            Some content here
        </column>
    </row>
</table>
Example Markdown
Here is the Table from above in Markdown (The spaces are optional)
<table>
    <row>
        <column width="50%">
            Title Cell of Column 1 | Some Value | Another Value
        </column>

        <column width="50%">
            Title Cell of Column 2 | Value spreads over entire row
        </column>
    </row>

    <row>
        <column width="25%">
            New Row this time
        </column>

        <column width="50%">
            And the Columns have changed in width
        </column>

        <column width="25%">
            Yet another Value
        </column>
    </row>
</table>
