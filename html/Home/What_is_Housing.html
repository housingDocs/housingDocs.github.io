What is Hypixel Housing?
An Overview

Hypixel Housing is a creative game mode on the popular Minecraft server, Hypixel, where players are given their own personal plots to build, design, and create almost anything they can imagine. It's a place where creativity meets community, allowing you to express yourself through construction and game design.

Connect to the server on Minecraft: Java Edition with the IP address mc.hypixel.net to join the fun!

Core Features
Your Personal Space
Every player receives their own plot of land to build on.
The default plot size is 31x31. You may expand your plot size with Mystery Dust or through the Hypixel Store.

Anything can be created in Housing as long as it follows Hypixel Rules.
Building & Creation
Block Library: Full access to blocks and items from Minecraft 1.8.x
Protools: Access Worldedit-like building tools for quick editing
Furniture: Place custom furniture and decorations

Interactive Elements
Spatial Triggers: Add Teleport Pads, Action Pads and Regions
Displays: Create Holograms and interactive NPCs
Logic Framework: Use Variables for advanced mechanics

Social & Community
Visit other players' houses
Show off your own creations
Get inspiration from what others have built

Getting Started

How to Access Your Housing Plot
1. Join Hypixel at mc.hypixel.net
2. Use /home to access your own plot
3. Start building your dream creation!

Quick Start Guide
Choose your first plot theme
Set up permissions in your House
Experiment with special blocks and items
Invite friends to visit your creation (/visitingrules)
