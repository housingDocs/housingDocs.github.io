Custom Commands
Create custom commands within the Housing
Introduction to Custom Commands
In every Housing, a set of commands such as /stuck and /clear is introduced to players.
However, you can add new commands to enhance user experience in your Housing.
Fundamentals
This feature contains multiple functions, so you can restrict certain commands, toggle command mode and its visibility in chat.
Below are the primary properties about this feature:
Limits
There are limited amount of custom commands you can create in your Housing.
  • Without Housing+: 10 Commands
  • With Housing+: 50 Commands
Command Mode
Command Mode has 2 options: Self and Targeted.
If the command is in Self mode, the command will execute on the player using the command when performed.
If the command is in Targeted mode, the command must include a target player as an argument, and runs on the target player when performed.
Required Group Priority
This function restricts a selected command so only players in a group with a required group priority can perform the command.
To learn more about Group Priority, head to Permissions and Groups page.
Command Listed
This function toggles whether a selected command should be listed to players in the commands auto-completion tab.
The command will only be listed to players in a group with a required group priority.
How to edit my custom commands?
To edit your custom commands, head into your Housing, and type in "/commands".

To add a custom command, click on Create Command (Marked as a paper)
To edit the actions in a command, left click on the selected command.
To edit the properties of a command, right click on the selected command.
To remove a command, right click on the selected command, then select Delete Command (Marked as a TNT).
