Event Actions
Many different methods of triggering actions using Event Actions
Introduction to Event Actions
This feature triggers actions when certain events (such as player joining/leaving) happen.
It can be accessed through:
Housing Menu (Ghast Tear/Nether Star) > "Systems" (Activator Rail) > "Event Actions" (Cobweb)
List of Events
Events
Trigger
Player Join
When a player joins the Housing
Player Quit
When a player leaves the Housing
Player Death
When a player dies
Player Kill
When a player kills another player
Player Respawn
When a player respawns
Group Change
When a player's group changes
PvP State Change
When a player's PvP state changes (from able to PvP to not, and vice versa)
Fish Caught
When a player catches something using a Fishing Rod while fishing
Player Enter Portal
When a player enters a Nether or End portal
Player Damage
When a player takes damage
Player Block Break
When a player tries to break a block (only if the player has no Build permission)
Start Parkour
When a player enters a Parkour with a Parkour Block
Complete Parkour
When a player finishes a Parkour with a Parkour Block
Player Drop Item
When a player drops an item
Player Pick Up Item
When a player picks up an item
Player Change Held Item
When a player changes their held item slot
Player Toggle Sneak
When a player holds or releases sneak
Player Toggle Flight
When a player starts or stops flying
