Functions
Cram multiple actions into one space using Functions
Access 
Housing Menu (Ghast Tear/Nether Star) > "Systems" (Activator Rail) > "Functions" (Activator Rail)
"/function" OR "/functions"
Introduction
A Function is used for triggering multiple actions at once. Functions behave similar to most other actions, able to be triggered using action triggers such as custom commands or Action Button/Pad.
Using Functions
A step-by-step tutorial on how to use Functions

[ i ] In the menu, click "Create Function" (Paper), and give your Function a name. After creation, the game brings you straight into the new Function.
[ ii ] Tell the game what you wish the Function to do when triggered, by clicking on "Add Action" (Paper), and selecting the desired action(s)
Steps below are optional.
[ iii ] Go back to the list of Functions, and right click the Function you wish to edit. Inside, you may rename, edit description and icon, delete, or set the automatic execution interval
Notes
"Automatic Execution" automatically trigger the function every N ticks (20 ticks = 1 second), with N being the number you set
Deleting the Function requires confirmation
