Custom Menus
Create and decorate interactive GUIs with Custom Menus
Introduction to Custom Menus

Custom Menus allows house owners to create their own menus (or GUIs as some people call them); you can edit the specific elements in a menu, allowing you to set the item in any slot and execute actions when they are clicked. Custom Menus also allow you to set the title and size of the menu displayed.

Note that there is a limit of 20 for non-Housing+ players; and 100 for Housing+ players.

You can create Custom Menus by:

Entering Housing Menu > Systems (Activator Rail) > Custom Menus (Chest) and creating a new menu by clicking the paper
&aSystems &7Manage regions, events, commands, &7and more for this house. &eClick to open!

>>>
&aCustom Menus &7Edit and create custom menus for &7use in your house. &eClick to edit!

>>>
&aCreate Menu
Using command "/menus" and creating a new menu by clicking the paper

/menus >>>
&aCreate Menu

Custom Menus Features

Change Title (Anvil) - Changes the Custom Menu's name. This will be where the "Chest" text would show up when opening the menu
Change Menu Size (Beacon) - Changes the menu size, this will change the size of the GUI. You can go from 1 - 6 rows
Edit Menu Elements (Ender Chest) - Shows you a template of your custom GUI, you can then edit it to your liking
Delete Menu (TNT) - Deletes the Custom Menu
Go Back (Arrow) - Go back to the Custom Menu selection menu
View Menu (Chest) - Shows you the finished GUI including title, items and executes actions when triggered.

You can click any empty slot to add an item (or stacks of one item). Make sure your choice is in your inventory!
You can right click an already occupied slot to change the item (or clear the slot) and left click to add an action that will trigger when the item is clicked in the interactive GUI.
You can use any action when clicking the GUI item. You can also add "Display GUI" action or "Close GUI" for menu linkage.

If you right click Edit Menu Elements in the main settings page, you can change items in freestyle.
