Introduction to Pro Tools
Pro Tools allow players to build or remove a large number of blocks quickly, they behave similarly to WorldEdit.

Different Pro Tools have different functions, with the "Region Selection Tool" being the core of all Pro Tools.
You may view (and take itemized) Pro Tools through:
Housing Menu (Ghast Tear/Nether Star) > "Pro Tools" (Stick)
