NPCs
Create interactive mobs or characters with NPCs
Introduction to NPCs
NPCs (Non-player characters) are a custom Housing item used to create mob-like characters anywhere in the house similar to the NPCs in lobbies used to select games.
They do not have any AI, but players are able to customize the skin of the NPC, items it has equipped, its name, left-click actions, right-click actions, and more settings depending on its entity type.
The NPC item is found under the Items tab in the Housing Menu.
Housing+ players can place 200 NPCs in a single world, while those without it can only place 80. (For more information on Housing+, check out this page.)

Upon spawning, it would display as a player with a Steve skin with one of the following default names:
Amarra, Aryn, Ayden, Baldrick, Cyne, Dene, Donald, Harry, Jaq, Jerry, Kenneth, Larry, Matt, Tane
Universal Settings for all Entity Types
Setting
Description
Possible Values
Rename NPC
Changes the nametag of the NPC.
Any string with a maximum of 32 characters
Left Click Actions
Edit the actions that run when a player uses left click on the NPC.
Any actions, refer to Edit Actions
Right Click Actions
Edit the actions that run when a player uses right click on the NPC.
Any actions, refer to Edit Actions
Look at Players
Toggle whether the NPC looks at the nearest player.
On / Off
Hide Name Tag
Toggle whethere the nametag of the NPC should be hidden.
On / Off
Additionally, in Left Click Actions menu, there is an addtional setting to bind it to Right Click Actions, which makes the NPC executes the same actions as long as the player clicks on the NPC.
Mob Settings
The NPC can be changed into a different variety of mobs, including:
Player, Pig, Sheep, Cow, Mooshroom, Chicken, Squid, Wolf, Ocelot, Iron Golem, Rabbit, Villager, Creeper, Zombie, Skeleton, Blaze, Spider, Cave Spider, Slime, Ghast, Zombie Pigman, Enderman, Silverfish, Magma Cube, Witch, Endermite, Snow Golem, Armor Stand
