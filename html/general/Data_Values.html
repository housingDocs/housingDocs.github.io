Data Values
Change block behavior using Data Values
What are Data Values?
Data Values are specific values that a block can have in Housing, which changes the blocks behavior when it's placed down. Data Values can be used to create a variety of blocks such as:
6-Sided Logs
Full blocks of a slab (Not double-slabs)
Blocks facing a specific direction
How to add Data Values?
To add a Data Value to a block, simply add "Data Value: (Some Number)" to the description/lore of a block. The numbers with their respective actions can be found in the Listt below.
