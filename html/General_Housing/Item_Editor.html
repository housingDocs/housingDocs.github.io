Item Editor
Customize an item's appearance and behaviour using the Item Editor
Introduction to the Item Editor
The Item Editor allows you to (nearly) fully customize an item's description and behaviour.
To access it, simply run "/item" or "/edititem".
When the command is executed, the Item Editor will open for the item currently held, and it is displayed on the second row of the menu.
Usage of the Item Editor
There are currently 6 editable item aspects:
Button
Action
Rename Item
As the name suggests, it allows you to change the name of the selected item (Formatting Codes can be applied)
Edit Lore
Change the description of the selected item (Formatting Codes can be applied)
Edit Enchantments
Apply or remove any enchantments onto the selected item (levels I~X)
Edit Actions
Customize what the selected item does when a player right clicks while holding it
Edit Flags
Toggle whether [ enchantments / modifiers (e.g. +7 Attack Damage) / "Unbreakable" / additional ] are shown in the description of the selected item
Unbreakable
Toggle whether the item's durability can drop
