Item Editor
Customize an item's appearance and behaviour using the Item Editor
Introduction to the Item Editor

The Item Editor allows you to (nearly) fully customize an item's description and behaviour.
To access it, simply run /item or /edititem.
When the command is executed, the Item Editor will open for the item currently held, and it is displayed on the second row of the menu.

If you are NOT the house owner, your group must have Item Editor permission enabled to use Item Editor. See Groups and Permissions for more details.

Usage of the Item Editor
There are currently 6 editable item aspects:

Button
Action
Rename Item
Allows you to change the name of the selected item (Formatting Codes can be applied)
Edit Lore
Changes the description of the selected item (Formatting Codes can be applied)
Edit Enchantments
Apply or remove any Minecraft 1.8 enchantments onto the selected item (Levels I~X)
Edit Actions
Customize what the selected item does when a player interacts with the item while holding it
Edit Flags
Toggle whether Item Information [e.g. Enchantments / Modifiers (e.g. +7 Attack Damage) / "Unbreakable" ] are shown in the description of the selected item
Unbreakable
Toggle whether the item's durability can drop
