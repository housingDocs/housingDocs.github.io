Introduction to Plot Size
Your Plot Size determines how much space you have to build and decorate your Housing. All Housings owned by your Minecraft account share the same Plot Size. Upgrading your plot allows for more creative freedom and larger builds.
How to Upgrade My Plot Size?
To upgrade your Plot Size, navigate to:

Housing Menu → House Settings → Plot Size Upgrade (marked with a Diamond Hoe).

Each Plot Upgrade has different requirements.
